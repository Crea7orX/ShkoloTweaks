Shkolo Tweaks


What is Shkolo Tweaks?

ShkoloTweaks is a web extension that modifies the content of the digital diary Shkolo.bg.
By modifing the content of the site, it adds features like a dark theme, widgets, stats, etc.

User Data

ShkoloTweaks does not collect any user data. The extension is open-source and can be found on GitHub.
In the future, the extension may collect some data with the user's consent.
